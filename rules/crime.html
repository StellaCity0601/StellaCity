🔫 犯罪ルール

犯罪・強盗・抗争に関するルール一覧です。
必ず確認したうえでRPを行ってください。

トラブル内容によっては、 運営判断により処罰対象となる場合があります。

🚨 基本ルール

  • 基本的にRPを遵守して行動してください。

⏰ サーバーリセット

  • リセット30分前〜リセット後15分以内は 全ての犯罪受注・開始禁止

⚔️ シーン終了条件

  • PD全滅
  • 犯人側全滅
  • PD降参
  • 犯人側逃走成功 → 指名手配中は延長される場合あり

👮‍♂️ PD関連

  • 白市民はPD指示に従うこと
  • PD視認状態でのアジト逃げ禁止
  • 犯罪中はPDへの警告なし発砲可能

📸 その他禁止事項

  • ダウン状態での報告禁止
  • 公務員を人質にする行為禁止
  • 人質がいる状態での撃ち合い開始禁止
  • 受注権は公平に話し合いで決定

💣 歪み（バグ）対応

1. PDへ報告し中止 → 罰金・押収・補填なし
→ 手荷物検査あり
→ 取得物は押収対象
2. 続行後クリップ提出 → 逃走成功で補填対象
→ 失敗時は通常通り罰金・押収・プリズン
※PDが来ない場合、15分待機で成功扱い

🔫 強盗・犯罪ルール

⚔️ 武器・戦闘

  • 準中型〜大型強盗は 即発砲OK → フリーカ・宝石強盗は人質任意
  • ゾンビ行為禁止
  • 防弾車両禁止（PDライオット除く）

🚁 特殊戦術

  • ヘリ撃ち（ヘリバイ）許可 → 一般市民巻き込み厳禁
  • ヘリアタック許可

🧍 人質関連

  • 人質は生存状態必須
  • 人質は見える位置へ配置 → 確認不可時はPD即突入可能
  • 犯人側発砲時、 全条件即解禁
  • 指名禁止 → セクハラ・トラブル防止

📋 進行ルール

  • 同組織複数受注禁止
  • 受注後30分以内開始必須
  • 犯罪中の車修理禁止
  • シーン中ガレージ使用禁止
  • 爆発物使用禁止
  • セーフゾーン逃げ込み禁止
  • 故意被せ禁止
  • 犯罪後現場戻り禁止
  • 物資隠し禁止
  • 合同時服装統一必須
  • エモート弾除け禁止
  • フリーカ・宝石除く 準中型〜大型強盗の人質禁止

🏥 EMS・蘇生

  • シーン終了までEMS治療・蘇生禁止

🧟‍♂️ ゾンビ行為

  • 原則禁止
